EXPENSE TRACKER

A simple web based expense tracker that helps you manage your 
daily expenses, set a monthly budget and see where your money 
is actually going.

I built this project to get better at HTML, CSS and JavaScript 
and to learn how to build something that actually feels useful.


What It Does

You can add an expense with a description, category, amount and 
date. All your expenses show up in a table and you can remove 
any of them. The total amount spent updates automatically every 
time you add or remove something.


Features

Add Expenses
Add any expense with a description, category, amount and date. 
Everything is saved in your browser so your data stays even 
after you close the tab.

Categories
Every expense belongs to a category — Food, Travel, Shopping, 
Bills, Entertainment or Other. Each category shows up as a 
colored label in the table so you can tell them apart at a glance.

Spending Chart
A donut chart on the left side of the page shows your spending 
broken down by category. It updates live every time you add or 
remove an expense so you always have a clear visual of where 
your money is going.

Monthly Budget
You can set a monthly budget using the Set Budget button. A 
progress bar shows how much of your budget you have used. It 
turns yellow when you cross 80% and red when you go over 100%. 
A warning message also appears at the top so you never miss it.

Export to CSV
One click on the Export CSV button downloads all your expenses 
as a spreadsheet file. You can open it directly in Excel or 
Google Sheets.

How To Use It

1. Open the file in any browser
2. Click "+ Add Expense" to add your first expense
3. Fill in description, category, amount and date
4. Click "Set Budget" to set your monthly spending limit
5. Watch the chart and progress bar update automatically
6. Click "Export CSV" to download your expenses anytime

Tech Used

- HTML
- CSS
- JavaScript
- Bootstrap 5
- Chart.js
- jQuery

What I Learned

This was my first time using Chart.js and I learned how to make 
a chart update in real time based on user input. I also learned 
how to use localStorage to save data in the browser without 
needing a backend or database. The budget warning feature taught 
me how to change styles dynamically based on conditions in 
JavaScript.
